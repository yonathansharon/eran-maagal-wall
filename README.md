# קיר התערוכה — מעגל 🟡

כלי אוצרות דו-ממדי (2D) לתכנון פריסת התערוכה ההיסטורית של **ער"ן**, סביב אוסף היצירות של האמן דוד (דודו) גרשטיין.

הכלי מאפשר לצוות לתכנן יחד את המייצג: להעלות צילומים אמיתיים של היצירות, לסדר אותם על קיר וירטואלי, לשנות גודל, ולראות הכול בקנה מידה אמיתי — לפני שנוגעים ביצירה פיזית אחת.

## מה אפשר לעשות

- **העלאת יצירות** — כמה תמונות בבת אחת; גוררים למקום ומשנים גודל (הנקודה בפינה).
- **"יצירה אבודה"** — הוספת מסגרת ריקה שמייצגת יצירה שעדיין לא אותרה. כך הקיר מציג גם את מה שנמצא וגם את מה שממתין לחזור.
- **קנה מידה אמיתי** — הזנת גובה הקיר (מ') ורוחב אמיתי לכל יצירה (ס"מ). מופיעים קו "גובה עיניים" (152 ס"מ) ודמות אדם לקנה מידה.
- **סיפור לכל יצירה** — שדה כותרת ושדה "הסיפור שמאחורי היצירה", לאיסוף סיפור-הסיפורים תוך כדי עבודה.
- **צבע קיר** — לבן גלריה / לבן חם / אפור / ירוק עמוק / כחול לילה / מותאם אישית.
- **שמירה, טעינה וייצוא PNG** — הפריסה נשמרת בדפדפן; ייצוא מפיק תמונה להטמעה במצגת או במייל.

## איך פותחים

פשוט פותחים את הקישור בדפדפן (ראו "פרסום" למטה), או פותחים את `index.html` מקומית בכל דפדפן.

> **פרטיות:** התמונות שמעלים נשמרות רק בדפדפן המקומי של כל משתמש (localStorage). הן **אינן** נשמרות במאגר ואינן מועלות לשום שרת. במאגר יש רק את קוד הכלי.

## פרסום כאתר חי (GitHub Pages)

1. ליצור מאגר (repository) חדש ולהעלות אליו את `index.html` ו-`README.md`.
2. **Settings → Pages → Source: Deploy from a branch → main / (root) → Save**.
3. אחרי דקה בערך, הכלי יהיה זמין בכתובת:
   `https://<שם-המשתמש>.github.io/<שם-המאגר>/`
